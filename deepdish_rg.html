§1.0 Project Overview

RadGotchi is a custom Pwnagotchi equipped with the face of Radbro. Running on a Raspberry Pi with BetterCAP for network monitoring, it learns from network behavior and adapts its personality based on activity patterns. A hacker's digital companion with community culture baked in.

This device performs passive WiFi monitoring using BetterCAP's deauthentication and handshake capture. Operate only on networks you own or have explicit authorization to test. Unauthorized use may violate federal law under the CFAA.

§5.1 Quick Start — Any Site

Drop these three things into your page:

Step 1 — Copy Sprites

Copy the assets/radbros/ sprite folder (15 PNGs) into your project root.

Step 2 — Include CSS + JS
<link rel="stylesheet" href="radgotchi.css"> <script src="radgotchi.js"></script>
Step 3 — Add Container Markup
<div id="radgotchi-container"> <img src="assets/radbros/HAPPY.png" class="radgotchi-face" id="radgotchi-face"> <span class="radgotchi-status" id="radgotchi-status">SIGINT NOMINAL</span> </div>

RadGotchi self-initializes on DOMContentLoaded, preloads all sprites, and starts running idle routines within 2 seconds. No config needed — just include and go.
