‚ñ∫ Key Features & Architecture

  • Real-time Eye Tracking: WebGL-based gaze visualization with crosshair positioning, scanning rings, and depth feedback
  • Live Camera Integration: Camera feed display with privacy mode toggle for user discretion
  • Dynamic Color System: Preset library + custom RGB controls with real-time render preview
  • Responsive HMI: Status bar with system indicators, connected user count, and external links
  • Multi-Tab Interface: Eye Control, Chat AI, Drone Fleet, and Control Panel modes
  • Privacy Controls: Built-in privacy mode for sensitive situations
  • Mobile Optimization: Fully responsive design for tablet and mobile viewing
  • Performance Monitoring: Frame rate display, connection status, and latency indicators

‚ñ∫ API Integration Points

  • DeepDish Eye API: Primary connection for eye tracking data and rendering commands
  • Prometheus Metrics: System health and performance monitoring
  • Grafana Dashboard: External link to comprehensive metrics dashboard
  • WebSocket: Real-time bidirectional communication for low-latency updates
